## Platforms Supported
- [x] iOS
- [x] Android


## Env. Requirement
```bash

$ node v18
$ react-native: "0.74.3"

```

## Installation
```bash
- Clone the Repository

$ git clone https://github.com/Vanush27/ToDo-Zustand.git
$ cd ToDoList


- Install Dependencies

npm install

```


## Running IOS app

```bash


# Install pod dependencies
$ pod-install

# Run on simulator
$ npm start ios

```

## Running Android app

```bash
# To use your device: 
# Connect your device
# Check if the device is running
$ adb devices
# Then
# Or without connecting Run on simulator
$ npm start android

```
